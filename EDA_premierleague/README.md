# Análisis Exploratorio de Datos (EDA) - Premier League 2021/2022

## Descripción
Análisis exhaustivo de los datos de la temporada 2021/2022 de la Premier League. 

## Objetivos
*   Limpieza y preparación de datasets de competiciones deportivas.
*   Visualización de métricas clave (goles, tarjetas, posesión).
*   Identificación de correlaciones entre el rendimiento de los equipos y las estadísticas de partido.
